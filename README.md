# Sobre
API RESTful de uma clínica médica, controlando informações de pacientes, médicos, prescrições e consultas. Projeto fullstack para estudos mas com foco maior no back-end.

## Principais Tecnologias
- Node.js
- Express
- Next
- MongoDB
- Mongoose
- Docker
- Multer
- Bcrypt
Para mais detalhes verificar [configurações](/MediApp/src/package.json).
